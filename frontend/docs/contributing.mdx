# Contributing

We welcome contributions to the Physical AI and Human-Aided Robotics educational resource. This guide explains how you can contribute to improve this curriculum.

## How to Contribute

### Documentation Improvements
- Fix typos and grammatical errors
- Improve explanations and examples
- Add missing content or clarify existing content
- Update outdated information

### Code Examples
- Add new practical examples
- Improve existing code samples
- Fix bugs in provided code
- Add alternative implementations

### Research Contributions
- Add research papers and academic resources
- Provide citations for existing content
- Contribute to the research bibliography

### Tutorials and Exercises
- Create new step-by-step tutorials
- Add assessment questions and exercises
- Develop practical assignments

## Getting Started

1. Fork the repository on [GitHub](https://github.com/fatimasheikh172/ai-native-book)
2. Create a new branch for your changes
3. Make your changes following the style guidelines
4. Submit a pull request with a clear description of your changes

## Style Guidelines

- Follow the existing documentation style
- Use clear, concise language
- Provide practical examples where possible
- Include appropriate citations
- Maintain the educational focus
